# 🕌 Tanzeem-e-Hussaini Official Website

A modern full-stack web application developed for **Tanzeem-e-Hussaini**, providing a centralized digital platform for Islamic content, events, announcements, galleries, videos, articles, prayer timings, and community engagement.

The application includes a secure **Admin Dashboard** for content management and a responsive public website for visitors.
